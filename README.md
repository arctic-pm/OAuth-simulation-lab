# 🔐 OAuth Attack Simulation Lab

An interactive cybersecurity project demonstrating how OAuth tokens can be used to access user data based on granted permissions.

## 🚀 Features
- Google OAuth Login
- Gmail API integration (read emails)
- Google Drive API (list + download files)
- Google Calendar API (view events)
- Base64 email decoding
- Interactive dashboard UI

## 🧠 Concept
This project demonstrates how:
- OAuth tokens act as delegated credentials
- Access is controlled by scopes
- Data can be accessed via APIs after user consent

## ⚠️ Disclaimer
This project is for educational purposes only. It only accesses data explicitly permitted by the user.

## 🛠️ Tech Stack
- Flask
- Google OAuth
- Gmail API
- Drive API
- Calendar API
